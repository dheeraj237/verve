# Tables and Blockquotes

This document demonstrates tables, blockquotes, and related formatting features. 

## Basic Tables

Create tables using pipes `|` and hyphens `-`:

| Name | Age | City | Country |
|------|-----|------|---------|
| Alice | 28 | New York | USA |
| Bob | 34 | London | UK |
| Charlie | 25 | Tokyo | Japan |
| Diana | 31 | Paris | France |

## Tables with Alignment

Use colons `:` to align columns:

| Left Aligned | Center Aligned | Right Aligned |
|:-------------|:--------------:|--------------:|
| Text | Text | Text |
| Content | Content | Content |
| Data | Data | Data |
| More | More | More |

## Tips

**Tables:**
- Use consistent spacing for readability
- Align columns with colons (`:`)
- Tables work best with short content
- Consider using HTML tables for complex layouts

**Blockquotes:**
- Great for highlighting important information
- Can contain any markdown elements
- Use nested quotes sparingly
- Add emoji or icons for visual appeal

**Pro Tip**: Click anywhere to toggle between preview and edit mode!
